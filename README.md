# Neha Gawali — Personal Portfolio

A premium, modern, and responsive personal portfolio website built with **Next.js**, **Tailwind CSS**, and **Framer Motion**. Featuring a dark-themed design with glassmorphism effects, smooth animations, and a modular data layer.

## 🚀 Key Features

- **Premium Design**: Dark-themed UI with warm orange gradient glows and glassmorphism cards.
- **Fully Responsive**: Optimized for desktop, tablet, and mobile viewing.
- **Animated Sections**: Smooth scroll reveals, staggered card animations, and interactive hover effects using Framer Motion.
- **Modular Data Layer**: Easily update personal info, projects, and experience via a single configuration file (`src/data/portfolio.ts`).
- **Production Ready**: Optimized for speed and scalability with Next.js App Router and standalone output support.

## 🛠️ Tech Stack

- **Framework**: Next.js 15+ (App Router)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Google Material Symbols & VectorLogoZone
- **Language**: TypeScript

## 🔧 Getting Started

### 1. Installation

```bash
npm install
```

### 2. Run Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the project locally.

### 3. Update Your Data

Open `src/data/portfolio.ts` and modify the `siteData` object to update your name, projects, experience, and other details. The changes will reflect instantly across the site.

## 📦 Deployment

This project is configured for easy deployment on **Render**, **Vercel**, or any static hosting provider.

To create a production build:

```bash
npm run build
```
